Projects

Splitcheck

Splitcheck is an iOS App for splitting bills between numerous people. It allows the user to calculate individual totals among a group fairly. It also supports tip calculation and saved contact lists.

I built and designed SplitCheck completely from scratch in Xcode using Swift. It is supported on all iPhones and is currently pending full release on the iOS App Store. 

Among the custom mechanisms developed in this project include contact structure savable as persisting data, realtime list multiselection, and UI/UX consistency extensions.

A machine vision update is underway that will allow the user to simply take a picture of their receipt and distribute the items among the group.
